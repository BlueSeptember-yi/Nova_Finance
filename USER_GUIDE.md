## 系统概述

NovaFinance 财务管理系统是一个完整的财务管理解决方案，涵盖会计科目、会计分录、采购销售、库存管理、银行对账和财务报表等核心功能。系统支持多公司运营，不同角色的用户拥有不同的操作权限。

### 用户角色

- **Owner（企业主）**: 拥有所有权限，可管理企业信息和所有业务数据
- **Accountant（会计）**: 负责会计科目、会计分录、财务报表等财务相关操作
- **Sales（销售）**: 负责客户管理、销售订单、收款等销售相关操作
- **Purchaser（采购）**: 负责供应商管理、采购订单、付款等采购相关操作
- **SuperAdmin（超级管理员）**: 系统级别管理员，可管理所有公司

## 登录系统

### 首次登录

1. 打开系统登录页面
2. 输入用户名和密码
3. 点击"登录"按钮
4. 登录成功后，系统会根据用户角色自动跳转到相应页面

### 登录后界面

登录成功后，您将看到系统主界面，包括：
- 顶部导航栏：显示当前用户信息和退出按钮
- 左侧菜单：根据用户角色显示相应的功能菜单
- 主内容区：显示当前页面的具体内容

## 工作台

工作台是系统的主页面，提供关键业务数据的概览。

### 功能说明

工作台显示以下信息：
- 财务概览：资产、负债、收入、支出等关键财务指标
- 业务统计：采购订单、销售订单、库存等业务数据统计
- 待办事项：需要处理的业务事项提醒
- 最近操作：最近的操作记录

## 基础数据管理

### 企业信息

**路径**: 企业信息

**功能说明**:
- 查看和编辑企业基本信息
- 设置企业规模、注册资本等信息
- 仅Owner角色可编辑

**操作步骤**:
1. 点击左侧菜单"企业信息"
2. 查看当前企业信息
3. 点击"编辑"按钮修改信息
4. 保存修改

### 会计科目管理

**路径**: 会计科目

**功能说明**:
- 管理企业的会计科目体系
- 支持多级科目结构
- 可参考标准会计科目创建科目
- 系统自动维护科目余额

**操作步骤**:

**查看科目列表**:
1. 点击左侧菜单"会计科目"
2. 查看科目树形结构
3. 可展开/折叠科目层级

**创建新科目**:
1. 点击"新建科目"按钮
2. 选择父级科目（可选）
3. 填写科目编码（公司内唯一）
4. 填写科目名称
5. 选择科目类型（资产、负债、权益、收入、费用、共同类）
6. 选择余额方向（借方/贷方）
7. 填写备注（可选）
8. 点击"保存"

**编辑科目**:
1. 在科目列表中点击"编辑"按钮
2. 修改科目信息
3. 注意：科目编码和类型修改需谨慎，可能影响已有数据
4. 点击"保存"

**参考标准科目**:
1. 系统内置国家标准会计科目
2. 创建科目时可参考标准科目的编码和名称
3. 标准科目为只读参考数据

### 会计分录

**路径**: 会计分录

**功能说明**:
- 创建和管理会计分录
- 支持手动录入和业务单据自动生成
- 过账前自动校验借贷平衡
- 可关联业务单据（采购、销售、付款、收款）

**操作步骤**:

**创建分录**:
1. 点击左侧菜单"会计分录"
2. 点击"新建分录"按钮
3. 填写记账日期
4. 填写摘要
5. 选择业务来源（可选）：采购订单、销售订单、付款、收款、手工录入
6. 添加分录明细：
   - 选择会计科目
   - 输入借方金额或贷方金额（至少一个不为0）
   - 填写备注（可选）
   - 点击"添加"按钮
7. 系统自动计算借方合计和贷方合计
8. 确保借贷平衡（借方合计 = 贷方合计）
9. 点击"保存"（保存为草稿状态）

**过账分录**:
1. 在分录列表中，找到需要过账的分录
2. 确认分录借贷平衡
3. 点击"过账"按钮
4. 系统自动校验借贷平衡
5. 过账成功后，分录状态变为"已过账"
6. 过账后，科目余额自动更新

4. 填写税号、银行账户等信息（可选）
5. 填写备注（可选）
6. 点击"保存"

### 客户管理

**路径**: 联系人管理 > 客户

**功能说明**:
- 管理客户基本信息
- 支持设置客户信用额度

**操作步骤**:

**添加客户**:
1. 点击"新建客户"按钮
2. 填写客户名称（必填）
3. 填写电话、邮箱、地址等信息
4. 填写税号（可选）
5. 设置信用额度（可选）
6. 填写备注（可选）
7. 点击"保存"

### 商品管理

**路径**: 商品管理 > 商品列表

**功能说明**:
- 管理商品基本信息
- 设置商品销售价和成本价
- SKU编码在公司内唯一

**操作步骤**:

**添加商品**:
1. 点击"新建商品"按钮
2. 填写SKU编码（必填，格式：2位字母+3位数字，公司内唯一）
3. 填写商品名称（必填）
4. 填写销售价（可选）
5. 填写成本价（可选）
6. 点击"保存"

### 采购管理

**路径**: 采购管理

**功能说明**:
- 创建和管理采购订单
- 支持多商品采购
- 支持折扣设置
- 订单过账后自动生成库存入库和会计分录

**操作步骤**:

**创建采购订单**:
1. 点击"新建采购订单"按钮
2. 选择供应商（必填）
3. 填写下单日期（必填）
4. 填写预计交货日期（可选）
5. 添加采购明细：
   - 选择商品（可选，也可手动输入商品名称）
   - 输入商品名称（必填）
   - 输入数量（必填）
   - 输入单价（必填）
   - 设置折扣率（默认1.0，即无折扣）
   - 系统自动计算小计
   - 点击"添加"按钮
6. 系统自动计算订单总金额
7. 填写备注（可选）
8. 点击"保存"（保存为草稿状态）

**过账采购订单**:
1. 在采购订单列表中，找到需要过账的订单
2. 点击"过账"按钮
3. 系统自动执行以下操作：
   - 生成库存入库记录（增加库存数量）
   - 更新商品加权平均成本
   - 生成会计分录（借：库存商品，贷：应付账款）
4. 过账后，订单状态变为"已过账"

### 销售管理

**路径**: 销售管理

**功能说明**:
- 创建和管理销售订单
- 支持多商品销售
- 支持折扣设置
- 支持多种收款方式
- 订单过账后自动生成库存出库和会计分录

**操作步骤**:

**创建销售订单**:
1. 点击"新建销售订单"按钮
2. 选择客户（必填）
3. 填写销售日期（必填）
4. 填写预计出库日期（可选）
5. 选择收款方式：现金、银行转账、赊账
6. 添加销售明细：
   - 选择商品（可选，也可手动输入商品名称）
   - 输入商品名称（必填）
   - 输入数量（必填）
   - 输入单价（必填）
   - 设置折扣率（默认1.0，即无折扣）
   - 系统自动计算小计
   - 点击"添加"按钮
7. 系统自动计算订单总金额
8. 填写备注（可选）
9. 点击"保存"（保存为草稿状态）

**过账销售订单**:
1. 在销售订单列表中，找到需要过账的订单
2. 点击"过账"按钮
3. 系统自动执行以下操作：
   - 生成库存出库记录（减少库存数量）
   - 计算销售成本（使用加权平均成本）
   - 生成会计分录（借：应收账款/银行存款，贷：主营业务收入；借：主营业务成本，贷：库存商品）
4. 过账后，订单状态变为"已过账"

## 库存管理

### 库存列表

**路径**: 库存管理 > 库存列表

**功能说明**:
- 查看所有商品的实时库存
- 显示库存数量和加权平均成本
- 库存数量由库存流水自动计算

**操作说明**:
1. 点击"库存管理" > "库存列表"
2. 查看所有商品的库存信息
3. 可搜索和筛选商品
4. 库存数量实时更新，反映最新的库存状态

### 库存流水

**路径**: 库存管理 > 库存流水

**功能说明**:
- 查看所有库存变化记录
- 记录每次入库和出库的详细信息
- 支持手工调整库存

**操作步骤**:

**查看库存流水**:
1. 点击"库存管理" > "库存流水"
2. 查看所有库存变化记录
3. 可筛选商品、日期、类型等条件
4. 每条流水记录包含：商品、类型（入库/出库）、数量、单位成本、来源、仓库位置等

**手工调整库存**:
1. 在库存流水页面点击"新建流水"按钮
2. 选择商品
3. 选择类型：入库或出库
4. 输入数量
5. 输入单位成本（入库时必填）
6. 选择来源类型：手工录入或调整
7. 填写仓库位置（可选）
8. 填写备注（可选）
9. 点击"保存"
10. 系统自动更新库存数量

**路径**: 银行对账 > 银行流水

**功能说明**:
- 录入和管理银行交易流水
- 用于与会计分录对账
- 系统会自动显示每条银行流水的对账状态（已对账/未对账）

**操作步骤**:

**录入银行流水**:
1. 点击"新建银行流水"按钮
2. 选择银行账户（必填）
3. 填写交易日期（必填）
4. 输入金额（必填）
5. 选择类型：收入或支出
6. 填写当前余额（可选）
7. 填写摘要（可选）
8. 点击"保存"

**批量导入银行流水（Excel）**:
1. 点击"导入Excel"按钮
2. 选择银行账户（必填）
3. 点击"下载导入模板"获取Excel模板
4. 按照模板格式填写银行流水数据
5. 点击"选择Excel文件并上传"
6. 系统自动解析并导入数据
7. 查看导入结果（成功/失败数量及错误信息）

### 银行对账

**路径**: 银行对账 > 对账

**功能说明**:
- 将银行流水与会计分录进行自动或手动匹配
- 支持一对一的匹配关系
- 自动筛选与银行存款相关的分录（排除采购成本、销售成本等无关分录）
- 支持查看已匹配对、未匹配银行流水、未匹配分录
- 生成银行存款余额调节表

**操作步骤**:

**进行对账**:
1. 点击"银行对账"菜单
2. 选择银行账户（必填）
3. 选择日期范围（必填）
4. 点击"查询"按钮，系统显示：
   - **已匹配**：显示已匹配的银行流水和分录对
   - **未匹配银行流水**：显示未匹配的银行流水
   - **未匹配系统分录**：显示未匹配的与银行存款相关的分录
5. 点击"自动匹配"按钮，系统自动匹配：
   - 金额相同（允许0.01误差）
   - 方向一致（收入/支出）
   - 日期相近（±3天）
6. 查看匹配结果，系统显示匹配成功的数量
7. 可以删除已匹配的对账记录（点击"删除"按钮）

**注意事项**:
- 系统只显示与银行存款科目（1002）及其子科目相关的分录
- 自动匹配会排除已匹配的记录
- 匹配成功后，可以在"已匹配"列表中查看
- 匹配成功后，银行流水的对账状态会自动更新为"已对账"
- 删除对账记录后，如果该银行流水没有其他对账记录，状态会自动更新为"未对账"
- 未匹配的银行流水和分录会分别显示在左右两列
